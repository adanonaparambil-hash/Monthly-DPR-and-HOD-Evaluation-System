# My Task Component

A comprehensive task management interface that matches the provided design specifications.

## Features

- **Active Timer Display**: Shows current active task with real-time timer
- **Task Board**: Organized view of all tasks with filtering capabilities
- **Status Tracking**: Visual indicators for task status (Not Started, In Progress, Completed)
- **Priority Management**: Color-coded priority indicators (High, Medium, Low)
- **Favorites Panel**: Quick access to frequently used tasks
- **Time Statistics**: Display of total paid hours and task logged hours
- **Calendar Widget**: Monthly calendar view with current date highlighting
- **Alert System**: Notifications for missing log entries

## Design Elements

- **Color Scheme**: Purple gradient header matching the provided design
- **Typography**: Modern, clean font hierarchy
- **Layout**: Three-column responsive layout (Favorites | Task Board | Actions)
- **Interactive Elements**: Hover effects, smooth transitions, and animations
- **Icons**: Font Awesome icons for consistent visual language

## Component Structure

- `my-task.component.ts`: Main component logic and data management
- `my-task.component.html`: Template with responsive layout
- `my-task.component.css`: Comprehensive styling matching the design

## Navigation

Accessible via: **DPR > My Task** in the main navigation menu.
